# Changelog

## v3.3.0 — Separate PDF Studio & Mobile Rebuild
- Moved PDF editing to a dedicated `editor.html` page.
- Added a separate “PDF Düzenleme” navigation section.
- Rebuilt the mobile navigation drawer and hamburger behavior.
- Added a fixed mobile application header.
- Rebuilt the PDF editor mobile layout with bottom navigation.
- Added mobile page, tool and properties panels.
- Improved dashboard cards, spacing and touch targets.


## v3.2.0 — Clean Dashboard
- Removed the large file-format icon strip from the hero area.
- Removed the decorative Import/Edit/Process/Export workflow.
- Added a prominent Ctrl+K tool search field.
- Reduced dashboard height and moved PDF tools higher on screen.
- Improved mobile header spacing.


## v3.1.0 — Editor Cleanup
- Removed the System Information panel from the dashboard.
- Simplified the PDF editor toolbar.
- Added a compact page/zoom/save navigation bar.
- Moved secondary functions under “Gelişmiş araçlar”.
- Added properties panel toggle and direct editor exit button.
- Improved workspace proportions and responsive behavior.


## 3.0.0 - Phase 3A

- Added browser-side Turkish and English OCR
- Added QR generation and QR reading
- Added local barcode reading where BarcodeDetector is supported
- Added heuristic PDF table extraction with CSV and JSON export
- Added indexed PDF search with page snippets
- Added explainable local document classification
- Added local operation history and JSON backup
- Preserved Phase 1 dashboard and Phase 2 PDF Studio

## v4.3 PDF Studio Pro
- Tarayıcı içinde OCR (Tesseract.js)

- Sayfa çoğaltma, sürükle-bırak sıralama ve silinen sayfaları geri getirme

- Ana sayfaya dönüş düğmesi
